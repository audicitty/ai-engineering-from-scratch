# Contributing to AI Engineering from Scratch

Thanks for wanting to make this resource better! Here's how you can help.

## Ways to Contribute

### 1. Fix Errors
Found a typo, wrong formula, or broken code? Open a pull request with the fix.

### 2. Improve Explanations
If something is confusing, you can:
- Rewrite it more clearly
- Add a better analogy
- Add a diagram or visual

### 3. Add Code Examples
Every concept should have runnable code. If you see a topic without code, add some!

### 4. Add Notes for New Topics
Check the roadmap for topics marked as "Planned" — pick one and write notes for it.

## Guidelines

- **Keep it simple.** Write as if explaining to someone who's never seen this before.
- **Use analogies.** "A tensor is like a box inside a box" beats "A tensor is a multi-dimensional array."
- **Include code.** Every concept should have a runnable example.
- **Test your code.** Make sure it actually runs before submitting.
- **Use consistent formatting.** Follow the structure of existing notes.

## How to Submit

1. Fork this repo
2. Create a branch: `git checkout -b fix/typo-in-transformers`
3. Make your changes
4. Commit: `git commit -m "Fix attention score formula in transformers notes"`
5. Push: `git push origin fix/typo-in-transformers`
6. Open a Pull Request

## Questions?

Open an issue with the "question" label and I'll respond.
